# ansible-cli-vbnet-hello-world

## Description
A POC for vbnet running on linux.
Built by ansible.

## Tech stack
- ansible
- vbnet
  - Framework 4.7
- mono 6.8
- bash

## Docker stack
- docker-cli
- ubuntu:20.04

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
